# Example 01: Basic Setup

This example demonstrates the simplest AI Agent Ecosystem setup without Ruler or Skills.

## Files Structure

```
01-basic-setup/
├── README.md (this file)
└── AGENTS.md (project-specific instructions)
```

## What's Included

- **AGENTS.md**: Basic agent instructions for the project
- No Ruler configuration
- No Skills
- Minimal setup for small projects

## Purpose

This setup is ideal for:
- Small projects
- Solo developers
- Quick prototypes
- Learning the basics

## Usage

Copy `AGENTS.md` to your project root:

```bash
cp AGENTS.md /your-project/AGENTS.md
```

## AGENTS.md Content

The AGENTS.md file includes:
- Project overview
- File structure
- Coding conventions
- Testing requirements
- Common commands

## Advantages

✅ Simple and straightforward  
✅ Low overhead  
✅ Easy to understand  
✅ Quick setup  

## Limitations

❌ No centralized rules  
❌ No reusable skills  
❌ Must duplicate for multiple projects  
❌ No organization-wide consistency
